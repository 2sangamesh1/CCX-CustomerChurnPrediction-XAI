# Customer Churn Prediction with Explainable AI

End-to-end churn prediction pipeline combining ML, LLM embeddings, SHAP explainability, and linear programming to generate actionable retention strategies.

## Tech

Python, XGBoost, scikit-learn, SHAP, MiniLM/BERT, PuLP

## Highlights

* Hybrid model (structured + text data)
* Explainable predictions (global + per-customer)
* Budget-optimized retention recommendations
* Best model: XGBoost (~0.90 AUC)

## Run

```bash
python app.py
```

## Dataset

IBM Telco Customer Churn Dataset
